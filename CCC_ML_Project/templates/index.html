Chapter 01 · The Investigation

What Is Crime Atlas?

Crime Atlas is a machine learning project that analyzes crime data across Indian cities — not to count incidents, but to find the invisible structure beneath them. It asks a different kind of question.

01 / THE QUESTION

Which cities are alike?

Not which city has the most crime — but which cities share the same behavioral fingerprint. That distinction changes everything about how crime is understood at scale.

02 / THE METHOD

The machine decides

No labels were assigned in advance. No categories were invented by analysts. An unsupervised algorithm was given the numbers and asked a single question: who belongs together?

03 / THE RESULT

A map that didn't exist

What emerged was a behavioral map of Indian urban crime — four distinct profiles, each with its own signature, surfaced entirely by the data itself.

Chapter 02 · The Problem

The Data Was Already Talking

Hidden in plain sight

Crime data across India accumulates continuously — reports filed, cases logged, incidents categorized. But raw volume is not intelligence. The deeper structure — which cities behave the same — was never visible.

Crime Atlas was built to surface what traditional analysis cannot: the behavioral geography of crime.

Chapter 03 · The Scope

A Different Kind Of Map

Crime Atlas does not count. It classifies. It does not report incidents — it discovers which cities share a crime identity. The difference is everything.

NOT THIS

Another crime statistic dashboard

There are many tools that chart incident counts, year-over-year trends, or city rankings. Crime Atlas is not one of them. Totals without structure tell you nothing about the shape of the problem.

THIS

A behavioral classification system

Crime Atlas uses K-Means clustering to group cities by behavioral similarity — across crime volume, weapon involvement, victim demographics, police deployment, and case resolution.

NOT THIS

Human-assigned categories

No analyst decided which city belongs in which group. No prior assumptions were built into the model. The groupings that emerged were entirely data-driven — the algorithm found them on its own.

THIS

Patterns the data chose for itself

The four behavioral profiles discovered by Crime Atlas were not invented. They were found — already present in the numbers, waiting for the right question to make them visible.
